ID e CLASS

CLASS

Existem 2 características principais do class

  • É possível utilizar o mesmo class em vários elementos diferentes
  • É possível utilizar várias classes no mesmo elemento

ID

Existem duas características principais do ID que são constrastantes com o class

  • Não é possível utilizar o mesmo ID em vários elementos diferentes
  • Não é possível utilizar VÁRIOS IDs no mesmo elemento

O ID funciona como um CPF. Não podem existir dois iguais e as mesma pessoa não pode ter vários CPFs
